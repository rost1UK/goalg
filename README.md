Goal Planner – Liquid Glass UI (Bootstrap)
Goal Planner is a modern and visually appealing web application designed to help users organize their tasks, goals, and daily plans efficiently. The interface is built using Bootstrap and enhanced with a stylish liquid glass (glassmorphism) design, creating a smooth, semi-transparent look with soft blur effects.
The application allows users to:
Create, edit, and delete goals
Organize tasks into categories or folders
Set deadlines and track progress
Filter and sort tasks easily
View statistics of completed and pending goals
The liquid glass design provides:
Frosted glass effect with backdrop blur
Smooth gradients and soft shadows
Light and dark theme support
Adaptive and responsive layout for all screen sizes
This project is ideal for showcasing modern UI/UX trends and can be easily deployed on any web server.

Планировщик целей – Liquid Glass дизайн (Bootstrap)
Планировщик целей — это современное веб-приложение для удобного управления задачами и целями. Интерфейс разработан с использованием Bootstrap и оформлен в стиле liquid glass (эффект стекла), создавая полупрозрачный, плавный и стильный внешний вид.
Возможности приложения:
Создание, редактирование и удаление задач
Организация целей по папкам и тегам
Установка сроков выполнения
Сортировка и фильтрация задач
Отображение статистики выполненных и активных целей
Особенности дизайна:
Эффект размытого стекла (blur + прозрачность)
Плавные градиенты и мягкие тени
Поддержка светлой и тёмной темы
Адаптивность под любые устройства
Проект отлично подходит для демонстрации современного UI/UX и может быть легко размещён на сервере.

Planificador de Objetivos – Diseño Liquid Glass (Bootstrap)
El Planificador de Objetivos es una aplicación web moderna diseñada para organizar tareas y metas de manera eficiente. Está construida con Bootstrap y utiliza un elegante diseño liquid glass (glassmorphism) que ofrece un efecto visual suave y translúcido.
Funciones principales:
Crear, editar y eliminar objetivos
Organizar tareas en carpetas o categorías
Establecer fechas límite
Filtrar y ordenar tareas fácilmente
Ver estadísticas de progreso
Características del diseño:
Efecto de vidrio con desenfoque (blur)
Gradientes suaves y sombras ligeras
Soporte para modo claro y oscuro
Diseño adaptable a cualquier dispositivo
Este proyecto es ideal para mostrar tendencias modernas de diseño UI y se puede desplegar fácilmente en cualquier servidor web.
